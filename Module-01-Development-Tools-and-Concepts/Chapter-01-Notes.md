# Chapter 1: Foundation (Week 1, Session 1)

## WSL Installation (Ubuntu)
**Step1** : Open Powershell and Type **wsl --version** or **wsl.exe --version** if only installed WSL installed.

**step2** : Update the WSL and See All the Linux Distributions :
```bash
wsl --version

wsl --list --online
```

**step 3** : Install Ubuntu in Windows : 
```bash
wsl --install ubuntu-24.04
```

**step 4** : To Create Password 
```bash
create default unix user account : ujwal
new password:
retypr new password:
```

Now We can use the Ubuntu in the Windows OS
