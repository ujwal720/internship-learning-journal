# Learnings

## Understanding Tools

* **Pandas** is essential for everyday data manipulation.
* **DuckDB** is highly efficient for large-scale analysis.
* **Excel** remains useful for quick statistical tasks and visualization.

## Using AI Effectively

* AI tools like ChatGPT can:

  * Generate code quickly
  * Explain complex logic
  * Speed up workflows

However:

* Always verify outputs
* Do not rely blindly on generated code

## Start with Exploration

* Never skip EDA
* Always understand:

  * Data structure
  * Missing values
  * Feature distributions

## Statistical Awareness

* Key principles:

  * Correlation does not imply causation
  * R² explains model fit
  * P-values indicate significance

## Managing Outliers

* Outliers require careful judgment:

  * Could be noise
  * Could be valuable signals

Use structured methods like IQR for detection.

## Importance of Visualization

* Visualization improves interpretability:

  * Heatmaps → patterns over time
  * Scatter plots → relationships
  * Box plots → spread and extremes

## Choosing the Right Approach

* Linear models → simple trends
* ETS models → seasonal data
* SQL tools → large datasets

## Spatial Thinking

* Geospatial analysis helps in:

  * Market planning
  * Store placement decisions
  * Competition analysis

## Efficiency Matters

* Prefer:

  * Parquet over CSV
  * DuckDB for heavy workloads

## Critical Mindset

* Always question results:

  * Are patterns real?
  * Is data reliable?

## Final Thought

Successful data analysis is not just about tools—it’s about combining technical skills with critical thinking to derive meaningful and actionable insights.
